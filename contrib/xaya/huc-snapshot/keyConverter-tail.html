Private Key Converter for the Playchain Huntercoin Snapshot

Using this tool, you can convert your Huntercoin (WIF) private key to the Playchain format. This allows you to import it into Playchain Core to claim the BITS balance granted in the Huntercoin snapshot.

This file is self-contained. If you want, you can download it, transfer it to an offline machine, and perform the conversion there. This provides maximum protection of your private keys.

This script is free software, released under the MIT license. The original source code is available on Github. The base58check processing is based on bs58check.
